autoLoadPHPClasses(string $className) : void

A custom autoloader function to dynamically load PHP classes. Converts the namespace and class name into a file path and requires the file if it exists. This allows for organizing the project with namespaces and eliminates the need for manual inclusion of each file.

Parameters
$className : string

The fully qualified name of the class being loaded.
